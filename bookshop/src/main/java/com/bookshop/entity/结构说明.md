# entity 层结构说明

## 1. 目录定位

`entity` 为数据库实体层，用于映射表结构字段。

## 2. 子目录说明

- `book`：图书实体对象。
- `user`：用户实体对象。

说明：`login` 暂无对应 `entity/login`；登录凭证或与 `user_account` 的关联表在接入 Spring Security + JWT 后再定。

## 3. 链路涉及

主要在 service 与 mapper 之间传递，不直接作为 controller 入参或出参。

## 4. 约束

- entity 与数据库字段保持一致，避免混入接口展示逻辑。
