# MySQL 数据库初始化

## 1. 目标

为 `Bookshop` 练手项目创建本地开发数据库，统一字符集和排序规则，避免后续出现乱码和排序不一致问题。

## 2. 当前环境检查结论

- 当前终端无法直接使用 `mysql` 命令（提示找不到命令）。
- 可能原因：
  - 未安装 MySQL 客户端；
  - 已安装但未将 `mysql.exe` 所在目录加入系统 `PATH`。

## 3. 版本与排序规则兼容策略

- 首选排序规则：`utf8mb4_0900_ai_ci`（MySQL 8.0 常用）
- 兼容降级：`utf8mb4_general_ci`（当 5.7 或低版本不支持 0900 时使用）

检查版本（可用时执行）：

```sql
SELECT VERSION();
```

## 4. 建库 SQL（推荐，MySQL 8+）

```sql
CREATE DATABASE IF NOT EXISTS `bookshop`
DEFAULT CHARACTER SET utf8mb4
DEFAULT COLLATE utf8mb4_0900_ai_ci;
```

## 5. 建库 SQL（兼容版）

```sql
CREATE DATABASE IF NOT EXISTS `bookshop`
DEFAULT CHARACTER SET utf8mb4
DEFAULT COLLATE utf8mb4_general_ci;
```

## 6. 创建开发账号与最小权限

> 初期建议先本机回环地址，后续再按部署方式调整主机白名单。

```sql
CREATE USER IF NOT EXISTS 'bookshop_dev'@'localhost' IDENTIFIED BY 'ChangeMe_123456';
GRANT ALL PRIVILEGES ON `bookshop`.* TO 'bookshop_dev'@'localhost';
FLUSH PRIVILEGES;
```

## 7. 验证步骤

```sql
SHOW DATABASES LIKE 'bookshop';
SHOW GRANTS FOR 'bookshop_dev'@'localhost';
```
