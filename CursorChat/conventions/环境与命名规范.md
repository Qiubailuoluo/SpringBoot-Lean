# 环境与命名规范

## 1. 环境命名

- `dev`：本地开发环境
- `test`：联调测试环境
- `prod`：生产环境

## 2. 数据库命名规范

- 数据库：`bookshop`
- 表名：小写下划线，例如 `book_info`、`user_account`
- 主键字段：统一 `id`（`BIGINT`）
- 时间字段：统一 `created_at`、`updated_at`

## 3. Java 包与类命名规范

- 根包建议：`com.bookshop`
- 包名全小写，按职责分层：
  - `controller`
  - `service`
  - `repository` 或 `mapper`
  - `entity`
  - `dto`
- 类名使用大驼峰，例如 `BookController`、`UserService`

## 4. 配置规范

- 默认配置文件：`application.yml`
- 环境拆分：
  - `application-dev.yml`
  - `application-test.yml`
  - `application-prod.yml`
- 当前默认激活：`dev`（通过 `application.yml` 中 `spring.profiles.active` 指定）
- 运行时可通过 `--spring.profiles.active=test` 或环境变量覆盖默认 profile。
- 敏感信息通过环境变量或本地私有配置注入，不提交仓库。

## 4.1 本机工具链记录（已确认）

- **注意**：Spring Boot 3 须用 **JDK 17+** 构建与运行。若系统默认 `java` 仍为 JDK 8，请在执行 `mvn` 前设置 `JAVA_HOME`（示例见 `CursorChat/springboot/01-项目初始化参数清单.md` §6.2）。
- JDK 路径：`D:/My_download/JDK17`
- JDK 版本：`17.0.15`
- Maven 主路径：`D:/My_download/java_MAVEN/apache-maven-3.6.3`
- Maven 版本：`3.6.3`
- Maven settings：`D:/My_download/java_MAVEN/apache-maven-3.6.3/conf/settings.xml`
- Maven 本地仓库：`D:/My_download/java_MAVEN/apache-maven-3.6.3/repo`

## 5. Git 提交建议

- 提交信息格式建议：`type(scope): message`
- 常用 `type`：
  - `feat`：新功能
  - `fix`：修复
  - `docs`：文档
  - `refactor`：重构
  - `test`：测试
