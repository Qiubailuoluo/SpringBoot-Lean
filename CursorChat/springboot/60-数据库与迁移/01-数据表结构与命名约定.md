# 数据表结构与命名约定

## 目标
统一当前项目表结构命名和 Java 字段映射规则。

## 代码与脚本位置
- `bookshop/src/main/resources/sql/book_info.sql`
- `bookshop/src/main/resources/sql/user_account.sql`
- `bookshop/src/main/resources/application.yml`

## 现状约定
- 表名使用下划线：如 `book_info`、`user_account`。
- Java 字段驼峰映射依赖 MyBatis 配置：`map-underscore-to-camel-case: true`。
- 密码字段使用 `password_hash`，禁止明文字段。

## 建议
- 时间字段统一 `created_at/updated_at`。
- 索引命名建议前缀化：`idx_`（普通索引）、`uk_`（唯一索引）。
