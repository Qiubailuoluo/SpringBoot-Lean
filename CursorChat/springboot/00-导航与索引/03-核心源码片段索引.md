# 核心源码片段索引

## 目标
提供“文档 -> 代码”的快速跳转索引，帮助你阅读时直接定位关键实现。

## 路径简写约定
- `java/...` 表示 Java 源码目录相对路径
- `resources/...` 表示资源目录相对路径

## 1) 应用启动入口
- 文件：`java/com/bookshop/BookshopApplication.java`
- 关注点：`SpringApplication.run(...)` 启动 Spring Boot 与嵌入式 Tomcat

## 2) 请求 Trace 链路入口
- 文件：`java/com/bookshop/common/trace/TraceIdFilter.java`
- 关注点：
  - 生成 `traceId`
  - 写入 `MDC`
  - 设置响应头 `X-Trace-Id`

## 3) Security 主配置
- 文件：`java/com/bookshop/config/security/SecurityConfig.java`
- 关注点：
  - `SessionCreationPolicy.STATELESS`
  - 白名单 `requestMatchers(...).permitAll()`
  - `addFilterBefore(jwtAuthenticationFilter, UsernamePasswordAuthenticationFilter.class)`

## 4) JWT 鉴权过滤器
- 文件：`java/com/bookshop/config/security/JwtAuthenticationFilter.java`
- 关注点：
  - 从 `Authorization` 解析 Bearer Token
  - 校验 token 类型是否为 access
  - 检查 Redis 黑名单并写入 `SecurityContext`

## 5) 全局异常处理
- 文件：`java/com/bookshop/exception/GlobalExceptionHandler.java`
- 关注点：
  - `BusinessException` -> 业务错误码响应
  - `MethodArgumentNotValidException` -> `VALIDATION_400`
  - `Exception` -> `SYSTEM_500`

## 6) 统一响应模型
- 文件：`java/com/bookshop/common/response/ApiResponse.java`
- 关注点：
  - 统一字段：`code/success/message/data/timestamp/path/traceId`
  - 工厂方法：`ok(...)`、`fail(...)`

## 7) 登录主流程
- 文件：`java/com/bookshop/service/login/impl/LoginServiceImpl.java`
- 关注点：
  - `login`：用户名查询 + BCrypt 校验 + 签发双 token
  - `refreshToken`：校验 refresh + 比对缓存
  - `logout`：access 加黑 + refresh 删除

## 8) Redis 令牌缓存
- 文件：`java/com/bookshop/service/login/TokenCacheService.java`
- 关注点：
  - `auth:refresh:{username}`
  - `auth:blacklist:{jwtId}`

## 9) 请求入口控制器
- 文件：
  - `java/com/bookshop/controller/book/BookController.java`
  - `java/com/bookshop/controller/login/LoginController.java`
  - `java/com/bookshop/controller/user/UserController.java`
- 关注点：Controller 层只做入参/响应，业务逻辑下沉 Service

## 10) 数据访问与 SQL
- 文件：
  - `java/com/bookshop/mapper/book/BookMapper.java`
  - `java/com/bookshop/mapper/user/UserMapper.java`
  - `resources/sql/book_info.sql`
  - `resources/sql/user_account.sql`
- 关注点：MyBatis 注解 SQL 与表结构对应关系

## 11) 配置与日志
- 文件：
  - `resources/application.yml`
  - `resources/application-dev.yml`
  - `resources/application-test.yml`
  - `resources/application-prod.yml`
  - `resources/logback-spring.xml`
- 关注点：profile 差异、JWT 参数、Redis 配置、日志 traceId pattern
