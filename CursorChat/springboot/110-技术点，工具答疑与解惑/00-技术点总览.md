## 技术点清单（第一版，仅作索引）
1. **Java Web 体系**  
	1. Tomcat（嵌入式容器、请求入口；规范写法为 *Tomcat*）  
	2. Filter 过滤器链（含常见安全链形态）  
	3. Servlet 规范与 Spring MVC（`DispatcherServlet`、Controller 路由）  
	4. Spring Boot（自动装配、嵌入式 Tomcat、与 Spring MVC 的组合方式）  
2. **数据库体系**  
	1. MyBatis（Mapper 与 SQL 映射）  
	2. MySQL（关系型存储；**YAML + Maven 依赖 + 建库** 一体看）  
	3. Flyway（模式迁移与版本化）   
3. **安全体系**（Spring Security + JWT + Redis，**独立见** [安全体系/00-本章导读.md](./安全体系/00-本章导读.md)）  
	1. Spring Security 链、白名单、无会话  
	2. JWT 结构、验签、解析、双令牌  
	3. Redis 键设计（refresh、黑名单、改密 等）  
	4. 结构图见该子目录 00/01/02/03
4. **可观测与联调体系**（ELK + Filebeat + Swagger/Postman + TraceId + 异常统一处理，见 [可观测与联调体系/00-本章导读.md](./可观测与联调体系/00-本章导读.md)）  
	1. ELK 与 Filebeat 协作  
	2. Swagger/OpenAPI 与 Postman 联调  
	3. 日志与 TraceId  
	4. 异常与统一处理（`@ControllerAdvice`）  
5. **待扩展**  
	1. 消息队列（如 RabbitMQ）  
	2. 异步、定时任务等
